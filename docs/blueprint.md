# **App Name**: ReflectFlow

## Core Features:

- Action Recording: Record a wide array of browser actions (clicks, typing, navigation, form submissions, scrolling).
- Material Design Overlay: Provide an overlay UI adhering to Material Design 3 Expressive guidelines.
- Step Editor: Offer an editable view of the recorded steps within the UI.
- Selective Playback: Play back recorded steps in their entirety, or play back from a user-selected step.
- Pause Recording: Pause recording.
- Dynamic Element Identification: Dynamically identify element via popup on hover (prioritizing ID, CSS selector, XPath).
- Local Storage: Save recorded sessions locally.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) for a calm, reliable feel.
- Background color: Light gray (#F0F2F5) for a clean and neutral base.
- Accent color: Soft amber (#FFB300) to highlight important actions, providing contrast and warmth.
- Body and headline font: 'Inter' (sans-serif) for a clean, modern, neutral design.
- Material Design icons that are intuitive and consistent throughout the extension.
- A clear and structured layout that aligns with Material Design expressive guidelines to enhance user experience.
- Subtle transitions and animations that provide feedback on user actions without being distracting.
